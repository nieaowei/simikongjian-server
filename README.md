# 一个简单的SSM项目Demo

本项目是一个Spring+SpringMVC+Mybatis+MySQL的一个小项目，项目只有简单的增删改查。
前端用了`JQuery`和`Boostrap`（其实也没用啥）。
纯粹是为了练习一下框架整合。
版本：`Spring（4.3.6）`、`SpringMVC（4.3.6）`、`Mybatis（3.4.2）`、`MySQL（5.7.0）`
虽然界面是食堂管理系统，但是，这都不是关键。。毕竟想改成啥就是啥。

开发环境：`Eclipse`+`Tomcat8.5`
